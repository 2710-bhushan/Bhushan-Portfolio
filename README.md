# ⚙️ Installation & Setup

## 📥 Clone Repository

```bash
git clone https://github.com/your-username/portfolio.git
```

---

## 📂 Navigate To Project

```bash
cd portfolio
```

---

## 📦 Install Dependencies

```bash
npm install
```

---

# ▶️ Run Development Server

```bash
npm run dev
```

---

# 🌐 Open In Browser

```bash

```

# ☁️ Deployment Guide

## Deploy On Vercel

### 1️⃣ Push Project To GitHub

```bash
git add .
git commit -m "Portfolio Website"
git push origin main
```

---

### 2️⃣ Open Vercel

👉 https://vercel.com/

---

### 3️⃣ Import GitHub Repository

- Login with GitHub
- Select repository
- Click Deploy 🚀
